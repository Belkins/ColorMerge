How to Play

Combine adjacent cells of the same color to score points. Avoid obstacles and use transformations to your advantage!

  1. Use the arrow keys to slide tiles in six directions on the hexagonal grid.
  2. Merge tiles of the same color to create a new color in the sequence or a slightly different hue if using a color spectrum.
  3. When two adjacent tiles with the same color collide, they merge into a single tile featuring the next color in the sequence.
  4. After each move, a new colored tile appears on the grid in a random empty cell.
  5. The game ends when there are no empty cells left for new colored tiles to appear, and no matching colors are left on the grid.
  6. The primary goal of the game is to turn all tiles on the grid into white tiles by strategically sliding and merging colored tiles.
